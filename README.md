# PyFry :smile::fire:

**Pyfry is a deep fry meme Image generator written in python and powered through OpenCV and DLib.**

# Deps
* python3x
* opencv (pip install opencv-python)
* imutils (pip install imutils)
* cmake (pip install cmake)
* dlib (pip install dlib)

# To run
* python PyFry.py
